Gets a value from the config

## Usage
```shell
> config get <get> {flags} 
 ```

## Parameters
* `<get>` value to get from the config

## Flags
* -h, --help: Display this help message

## Examples
  Get the current startup commands
```shell
> config get startup
 ```
