Applies the square root function to a list of numbers

## Usage
```shell
> math sqrt {flags} 
 ```

## Flags
* -h, --help: Display this help message

## Examples
  Apply the square root function to a list of numbers
```shell
> echo [9 16] | math sqrt
 ```
