Displays the last 100 commands.

## Example

```shell
> history
─────┬────────────────────────────────────────────────────────────────────────
  #  │
─────┼────────────────────────────────────────────────────────────────────────
...
 97  │ date
 98  │ ls
 99  │ ls -la
─────┴────────────────────────────────────────────────────────────────────────
```
